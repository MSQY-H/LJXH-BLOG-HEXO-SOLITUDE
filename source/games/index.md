## Minecraft

{% img /img/mc.png mc %}

- 简介：Minecraft（中文名《我的世界》）是一款由Mojang Studios开发的3D沙盒类电子游戏，玩家可以在方块构成的世界中自由创造、探索和生存。
- 评价：好玩，爱玩！
- 星级：★★★★★

## Forza Horizon 4

{% img /img/horizon.jpg 地平线4 %}

- 简介：《极限竞速：地平线4》是由Playground Games开发的一款赛车竞速游戏，为《极限竞速：地平线》系列的第四部作品，于2018年10月2日发行。
- 评价：风景好看！
- 星级：★★★★★

## Cities: Skylines

{% img /img/cities.webp 天际线 %}

- 简介：《都市：天际线》是由Colossal Order开发、Paradox Interactive发行的一款模拟经营游戏，于2015年3月11日发行。2026年初，游戏所有后续开发工作由Iceflake Studios全面接手。
- 评价：😋
- 星级：★★★★★

## CS2

{% img /img/cs2.webp cs2 %}

- 简介：《反恐精英2》是由Valve公司开发的第一人称射击游戏，采用起源2引擎制作，于2023年9月28日通过Steam平台和完美世界平台正式发行，为《反恐精英：全球攻势》的续作。
- 评价：不错，但是我技术太菜了
- 星级：★★★★★

## 塞尔达传说：旷野之息

{% img /img/zelda1.webp 旷野之息 %}

- 简介：《塞尔达传说 旷野之息》是任天堂企划制作本部开发、Monolith Soft协力开发的动作冒险游戏，为《塞尔达传说》主系列的第19作
- 评价：好玩，自由度太高了
- 星级：★★★★★

## 塞尔达传说：王国之泪
